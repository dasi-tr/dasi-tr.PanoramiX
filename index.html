PanoramiX

Capture any scrollable area in Windows and turn it into one continuous image. Works with browsers, IDEs, Terminal, CMD, logs, chats and more.

Any scrollable window

Select an area, scroll and let PanoramiX combine the result.

How it works

  1. Open PanoramiX.
  2. Select the area you want to capture.
  3. Scroll through the content.
  4. Press Enter or Finish.
  5. The complete image is copied to your clipboard.
Windows security notice

PanoramiX is currently in beta and is not digitally signed yet. Windows may display a warning on first launch.

In that case, select: More info → Run anyway

PanoramiX בעברית

PanoramiX הוא כלי ל־Windows שמצלם אזור ארוך שאפשר לגלול בו והופך אותו לתמונה אחת רציפה.

הוא עובד בדפדפנים, בסביבות פיתוח, ב־Terminal, ב־CMD, בלוגים, בצ׳אטים ובחלונות נוספים שניתן לגלול בהם.

איך משתמשים?

  1. פותחים את PanoramiX.
  2. מסמנים את האזור שרוצים לצלם.
  3. גוללים עד סוף התוכן.
  4. לוחצים על Enter או Finish.
  5. התמונה המלאה מועתקת אוטומטית ללוח ההעתקה.
