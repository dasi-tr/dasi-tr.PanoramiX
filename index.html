How it works

  1. Open PanoramiX.
  2. Select the area you want to capture.
  3. Scroll through the content.
  4. Press Enter or Finish.
  5. The complete image is copied automatically to your clipboard.

Frequently asked questions

Answers about using PanoramiX for scrolling screenshots in Windows applications.

What is PanoramiX?

PanoramiX is a Windows scrolling screenshot tool. It captures a selected scrollable area and combines the content into one continuous image.

Does PanoramiX work outside the browser?

Yes. PanoramiX works with scrollable areas in Windows applications, including IDEs, Terminal, CMD, logs, chats and browsers.

Can I capture long Terminal or CMD output?

Yes. Select the Terminal or CMD area, scroll through the output, and PanoramiX combines it into one image.

Can I take a scrolling screenshot inside an IDE?

Yes. PanoramiX can capture scrollable code editors, development tools and other Windows application areas.

Is PanoramiX free?

The current PanoramiX beta is available as a free download for Windows.

Why does Windows show a security warning?

The current beta is not digitally signed yet, so Windows may display a warning during the first launch.
