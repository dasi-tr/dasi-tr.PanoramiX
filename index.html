How it works

  1. Open PanoramiX.
  2. Select the area you want to capture.
  3. Scroll through the content.
  4. Press Enter or Finish.
  5. The complete image is copied automatically to your clipboard.
Windows security notice

PanoramiX is currently in beta and is not digitally signed yet. Windows may display a warning on first launch.

In that case, select: More info → Run anyway

איך משתמשים?

  1. פותחים את PanoramiX.
  2. מסמנים את האזור שרוצים לצלם.
  3. גוללים עד סוף התוכן.
  4. לוחצים על Enter או על Finish.
